# NAO Robot Control App

A comprehensive application for controlling and interacting with NAO robots, built with Python, TypeScript, and C++.

## Overview

This repository contains a full-stack application for NAO robot control and management. The project leverages multiple programming languages to provide a robust backend, responsive frontend, and optimized robotic control systems.

## Tech Stack

- **Python** (69.7%) - Core robot control logic and backend services
- **TypeScript** (28%) - Frontend application and type-safe utilities
- **C++** (1.2%) - Performance-critical robotic operations
- **JavaScript** (1.1%) - Web utilities and scripting

## Features

- Robot connection and communication management
- Real-time control interface
- Motion planning and execution
- Sensor data monitoring
- Telemetry and logging

## Getting Started

### Prerequisites

- Python 3.8+
- Node.js and npm
- NAO SDK
- C++ compiler (for C++ components)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sushanthsujeerkumar/nao-robot-control-app.git
   cd nao-robot-control-app
   ```

2. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Install Node.js dependencies:
   ```bash
   npm install
   ```

4. Build C++ components:
   ```bash
   mkdir build && cd build
   cmake ..
   make
   ```
